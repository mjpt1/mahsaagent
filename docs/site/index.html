Mahsaagent

جعبه ابزار فارسی برای ادیتور و اپ — RTL، جلالی، اعتبارسنجی ایرانی، geo رسمی، بانک، MCP چندکلاینته.

نصب

git clone https://github.com/mjpt1/mahsaagent.git
cd mahsaagent
npm install && npm run build
node dist/index.js doctor
node dist/index.js init-mcp

تا قبل از publish روی npm از کلون گیت استفاده کنید. نسخهٔ فعلی: ۰٫۸٫۱ · ۶۴ ابزار

ابزارها

  • stdio: node dist/index.js serve
  • HTTP: node dist/index.js serve-http --port 3847 (برای تونل: MAHSAAGENT_TOKEN)
  • کلاینت‌ها: docs/clients.md · امنیت: SECURITY.md

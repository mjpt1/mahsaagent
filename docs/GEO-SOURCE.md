# دادهٔ تقسیمات کشوری

فایل‌های `src/data/official/*.json` از مجموعهٔ [ahmadazizi/iran-cities](https://github.com/ahmadazizi/iran-cities) نسخهٔ v3.0 تبدیل شده‌اند.

برای بازتولید:

```bash
# CSVها را در tmp-iran-cities بگذارید، سپس:
npm run import:geo
```

مجوز و شرایط استفاده از داده را در مخزن بالادستی ببینید.
